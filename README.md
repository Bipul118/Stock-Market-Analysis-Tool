# 📈 Stock Market Analysis Tool

An AI-powered stock market analysis dashboard built with Python.

## Features

✅ Live Stock Data  
✅ Technical Analysis  
✅ MA50 / MA200 Indicators  
✅ RSI Indicator  
✅ BUY / SELL Signal  
✅ ML Price Prediction  
✅ Portfolio Tracker  
✅ Interactive Dashboard  

## Tech Stack

- Python
- Pandas
- NumPy
- Streamlit
- Plotly
- Scikit-learn
- Yahoo Finance API
